# Characters and how to type them 

- `|` - Pipe (vertikal bar)
  - Mac:  `option` + `i`
  - Windows: `AltGr` + `´`

- `{` - Curly brackets open (tuborg klammer)
  - Mac: `option` + `shift` + `8`
  - Windows: `AltGr` + `7`

- `}` - Curly brace close
  - Mac: `option` + `shift` + `9`
  - Windows: `AltGr` + `0`
- `[` - Square brackets open (firkantede klammer)
  - Mac: `option` + `8`
  - Windows: `AltGr` + `8`
- `]` - Square brackets close (firkantede klammer)
  - Mac: `option` + `9`
  - Windows: `AltGr` + `9`
- `\` - Backslash
  - Mac:  `option` + `shift` + `7`
  - Windows: `AltGr` + `<`
- `$` - Dollar sign
  - Mac: `option` + `shift` + `3`
  - Windows: `altGr` + `4`
- `` ` `` - Backtick
  - Mac: `shift` + `´`
  - Windows: `shift` + `´`
